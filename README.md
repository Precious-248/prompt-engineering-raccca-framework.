# RACCCA Prompt Engineering Project

The difference between a chatbot and a research assistant is often in the structure of the prompt. I used the **RACCCA framework** (Role, Act, Context, Constraints, Communication Style, Audience) to transform a generic request into a professional, high-quality result.

As an aspiring Data Analyst, I am finding structured prompting essential for organizing analytical tasks, generating clearer insights, and improving the quality of reports and documentation. This experience reinforced an important lesson: the quality of the output is directly proportional to the quality of the instructions.
